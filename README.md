# TI-Root
A program written for the TI 84 Plus CE that can solve quadratic equations in form ax^2 + bx + c. This is a work in progress with several improvements already planned. To use download the QUAD.8xp from the releases section and copy the QUAD.8xp file to your TI 84 Plus CE using ethier TI connect CE, TILP, or TIcalc.link. Once copied over run by pressing "prgm" and selecting QUAD. NOTE: If you are updated past 5.4.1, (such as myself) use the Artifice jailbreak found [here](https://yvantt.github.io/arTIfiCE/) to run.

Usage:
The calculator will prompt you to type in A=, B=, C=. Once you do hit enter and the solutions (or real roots) will appear. After the roots have been provided press "clear" to exit the program or "2nd" to enter a new equation.

Warning:
This program has a massive bug that you may run into if you try to open a system menu while the program is running. The issue has been identified and hopefully will be fixed soon, thank you for you patience.

Building:
Download the CE programming toolchain. run make while in the source code directory and it should build successfully.
